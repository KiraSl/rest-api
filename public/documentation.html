Movies API

API Overview

Provides data about the Soviet and Russian movies.

MOVIES

GET /movies

Get the list of all available movies.

Request Parameters

None

Response Data

A JSON object holding data about all the movies.

GET /movies/:Title

Get data about a movie by title.

Request Parameters

Title

Response Data

A JSON object holding data about a movie (title, description, year of issue, genre, director, image URL).

Example: { "Genre": { "Title": "Drama", "Description": "These are serious stories that hinge on events that regularly happen in everyday life. They usually focus on character and how these people act over time. Drama is regularly mashed up with other genres because most movies and tv rely on character-driven stories to keep the audience involved." }, "Director": { "Name": "Andrey Zvyagintsev", "Born": "1964", "Death": "", "Bio": "Andrei Zvyagintsev is a Russian filmmaker." }, "_id": "5d69aab363e083d58db959b5", "Title": "Leviathan", "Description": "In a Russian coastal town, Kolya is forced to fight the corrupt mayor when he is told that his house will be demolished. He recruits a lawyer friend to help, but the man's arrival brings further misfortune for Kolya and his family.", "Year": "2014", "ImageURL": "", "Featured": true }

GET /genres/:Title

Get data about a movie genre by title.

Request Parameters

Title

Response Data

A JSON object holding data about a movie genre (title, description).

Example: { "Title": "Cartoon", "Description": "A cartoon is a type of illustration, possibly animated, typically in a non-realistic or semi-realistic style. The specific meaning has evolved over time, but the modern usage usually refers to either: an image or series of images intended for satire, caricature, or humor; or a motion picture that relies on a sequence of illustrations for its animation." }

GET /directors/:Name

Get data about a movie director by name.

Request Parameters

Name

Response Data

A JSON object holding data about a director (name, birth year, death year, bio)

Example: { "Name": "Mark Zakharov", "Born": "1933", "Death": "", "Bio": "Mark Zakharov was born on 13 October 1933 in Moscow, Soviet Union. He has been the Artistic Director of Moscow's Lenkom Theatre since 1973, where he has helped to define the landscape of Moscow's theatrical culture." }

USERS

POST /account

Allows to register a new user.

Request Body

JSON Object

Example: { "Username" : "annasmith", "Password" : "sdfbsdhjb56", "Email": "annasm17@gmail.com" }

Response Data

JSON Object

Example: { "FavoriteMovies": [], "_id": "5d6c15b6e9fdb22759a6548d", "Username": "annasmith", "Password": "sdfbsdhjb56", "Email": "annasm17@gmail.com", "__v": 0 }

PUT /account/:id

Allows a user to update their account information.

Request Parameters

id

Request Body

JSON Object

Example: { "Email": "ivanpetrov@mail.ru" }

Response Data

A JSON object with the data about the user.

Example: { "FavoriteMovies": [], "_id": "5d69b94625e8682afb8e67a6", "Username": "ivanpetrov", "Password": "ip2019", "Email": "ivanpetrov@mail.ru", "Birthday": "1997-06-23T00:00:00.000Z" }

GET /account/:id/favorite/movies

Get the user's favorite movies.

Request Parameters

id

Response Data

An array with the movies IDs.

Example: [ "5d69b58225e8682afb8e67a3", "5d69b0de63e083d58db959b8" ]

POST /account/:id/favorite/movies/:movieid

Adds a movie to the user's list of favorite movies.

Request Parameters

movieid

Response Data

A JSON object with the data about the user.

{ "FavoriteMovies": [ "5d69b58225e8682afb8e67a3", "5d69b0de63e083d58db959b8" ], "_id": "5d69b7ab25e8682afb8e67a4", "Username": "Kira", "Password": "Abch85", "Email": "ab@gmail.com", "Birthday": "1985-01-25T00:00:00.000Z" }

DELETE /account/:id/favorite/movies/:movieid

Removes a movie from the user's list of favorites.

Request Parameters

id, movieid

Response Data

A JSON object with the data about the user.

Example: { "FavoriteMovies": [], "_id": "5d69b94625e8682afb8e67a6", "Username": "ivanpetrov", "Password": "ip2019", "Email": "ivanp@yandex.ru", "Birthday": "1997-06-23T00:00:00.000Z" }

DELETE /account/:id

Deletes the user's account.

Request Parameters

id

Response Data

A text message with the confirmation that the account was successfully deleted.
