Oscar Movies API

API Overview

Provides access to the information related to the movies which won in the "Best Picture" Oscar nomination.

MOVIES

GET /movies

Get the list of all available movies.

Response Data

json: [{ title: string, dateOfIssue: integer, directors: array[object], genres: array[string], description: string }, { title: string, dateOfIssue: integer, directors: array[object], genres: array[string], description: string } ]

Example: [{ title: 'Green Book', dateOfIssue: 2018, directors: [ { name: 'Peter Farrelly', bio: '', birthYear: 1956, deathYear: null, } ], genres: [ 'Biography', 'Comedy', 'Drama' ], description: 'A working-class Italian-American bouncer becomes the driver of an African-American classical pianist on a tour of venues through the 1960s American South.' }, { title: 'The Shape of Water', dateOfIssue: 2017, directors: [ { name: 'Guillermo del Toro', bio: '', birthYear: 1964, deathYear: null, } ], genres: [ 'Adventure', 'Drama', 'Fantasy' ], description: 'At a top secret research facility in the 1960s, a lonely janitor forms a unique relationship with an amphibious creature that is being held in captivity.' }]

GET /movies/{title}

Get data about a certain movie (description, date of issue, director, genre).

Request Parameters

title

Response Data

json: { title: string, dateOfIssue: integer, directors: array[object], genres: array[string], description: string }

Example: { title: 'Green Book', dateOfIssue: 2018, directors: [ { name: 'Peter Farrelly', bio: '', birthYear: 1956, deathYear: null, } ], genres: [ 'Biography', 'Comedy', 'Drama' ], description: 'A working-class Italian-American bouncer becomes the driver of an African-American classical pianist on a tour of venues through the 1960s American South.' }

GET /genres/{name}

Get data about a certain movie genre by it's name.

Request Parameters

name

Response Data

A text message with information about the genre.

GET /directors/{name}

Get data on a certain director (bio, birth year, death year).

Request Parameters

name

Response Data

json: { name: 'string', bio: string, birthYear: integer, deathYear: integer }

Example: { name: 'Peter Farrelly', bio: '', birthYear: 1956, deathYear: null, }

USERS

POST /account

Allows to register a new user.

Request Body

object

Example: { name: 'James Smith', username: 'jsm2019', password: 'strong password', email: 'somename@something.com', dateOfBirth: '12.05.1985' }

Response Data

object

Example: { id: 55, name: 'James Smith', username: 'jsm2019', password: 'strong password', email: 'somename@something.com', dateOfBirth: '12.05.1985' }

PUT /account/{id}

Allows the user to update their account information.

Request Parameters

id

Response Data

A text message with information that was updated.

GET /account/{id}/favorite/movies

Get information about the user's favorite movies.

Request Parameters

id

Response Data

json: [{ title: string, dateOfIssue: integer, directors: array[object], genres: array[string], description: string }, { title: string, dateOfIssue: integer, directors: array[object], genres: array[string], description: string } ]

POST /account/{id}/favorite/movies

Allows to add a movie to their list of favorites.

Request Parameters

id

Request Body

object

Example: { title: 'Green Book', dateOfIssue: 2018, directors: [ { name: 'Peter Farrelly', bio: '', birthYear: 1956, deathYear: null, } ], genres: [ 'Biography', 'Comedy', 'Drama' ], description: 'A working-class Italian-American bouncer becomes the driver of an African-American classical pianist on a tour of venues through the 1960s American South.' }

Response Data

object

DELETE /account/{id}/favorite/movies/{title}

Allows to remove a movie from the user's list of favorites.

Request Parameters

id, title

Response Data

A text message with information that the movie was removed from the list of favorites.

DELETE /account/{id}

Allows to delete the user's account.

Request Parameters

id

Response Data

A text message with the confirmation that the account was successfully deleted.
